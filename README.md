# PING-ME-BABY-Ekosistem-IoT-Berbasis-ESP32-untuk-Penderita-Alzheimer
Proyek ini adalah sistem intervensi psikologis otomatis menggunakan fitur Context-Aware Voice Interaction. Alat ini membantu penderita Alzheimer mengatasi fenomena sundowning dengan suara keluarga yang familiar dan fitur pemantauan lokasi real-time.
